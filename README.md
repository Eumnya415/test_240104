# test_240104
일일과제 : 학점관리 폼 작성

## 요구사항
### 회원관리 폼 제작
* 학점관리 폼 내용에 남 여 성별을 선택할 수 있는 항목을 만들되 남 또는 여자 글자만 선택해도 점이 표시되도록 함.
* 점수입력은 0~100점 이내의 숫자만 가능하도록 함.
* 입력버튼을 저장 버튼을 누를 경우 빈칸이 하나라도 있으면 경고창 표시
* 폼 시작과 함께 학번에 커서 위치
